# Indie Hacker Toolkit

## Overview
Pack of 8 Notion-ready markdown templates for solo SaaS founders covering launch, revenue tracking, content, feedback, pricing strategy, AI tools, customer acquisition, and competitor analysis. Sold at $19.

## Tech Stack
- **Format:** Markdown files designed for Notion import (no framework, no dependencies)
- **Distribution:** Git repo / digital download

## Key Commands
- No build or dev commands -- this is a content product
- Import into Notion: Settings > Import > Markdown > upload `.md` files

## Project Structure
- `templates/` -- 8 planning/business templates
  - `saas-launch-checklist.md` -- 120+ item checklist (pre-launch, technical, marketing, launch day, post-launch)
  - `revenue-tracker.md` -- Monthly MRR, churn, expenses, P&L, health indicators
  - `content-calendar.md` -- 30-day content plan across Twitter/X, Reddit, IH, blog
  - `customer-feedback-system.md` -- Feature requests, bug reports, interviews, NPS tracking
  - `pricing-strategy-worksheet.md` -- Competitor analysis, value metrics, 3-tier builder, experiments log
  - `ai-tool-stack-guide.md` -- Curated AI tools for solo founders with ROI tracking (2026)
  - `acquisition-playbook.md` -- 20+ customer acquisition channels ranked by effort/impact ($0 budget)
  - `competitor-analysis-framework.md` -- Systematic competitor intelligence with quarterly review system
- `README.md` -- Product overview, Notion import instructions, customization guide
- `GUMROAD_LISTING.md` -- Sales copy
